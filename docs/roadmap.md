# Roadmap

## Phase 1

- App bootstrap
- API skeleton for all feature domains
- Frontend console shell
- Rules Studio foundation
- Built-in template placeholders

## Phase 2

- SQLite schema and repositories
- User auth and sessions
- Nodes and subscriptions CRUD
- Proxy groups and rule persistence
- Template storage and rendering preview

## Phase 3

- Xray config renderer
- Config snapshots and diffs
- User-facing subscription output endpoints
- Traffic ingestion and dashboard metrics

## Phase 4

- Remote server registry
- Agent connection flows
- Streamed remote tasks for Xray and Nginx
- Certificate issuance and deployment
- DNS provider credentials and zone actions

## Phase 5

- Notifications
- Backups and restore
- Advanced settings
- UI refinement and search
